# wildlife_detection_system

## Overview

This web application detects wildlife animals from uploaded images using Amazon Rekognition.

## Features

- Upload wildlife image
- Animal detection
- Confidence score
- Detection history
- Cloud storage

## Technologies

- Python
- HTML
- CSS
- JavaScript
- AWS Rekognition
- DynamoDB
- S3

## AWS Services

- Rekognition
- S3
- Lambda
- API Gateway
- DynamoDB
